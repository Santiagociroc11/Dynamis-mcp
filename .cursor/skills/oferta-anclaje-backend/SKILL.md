# Oferta, Anclaje de Valor y Backend

Modelo para construir ofertas que convierten y suben el LTV. Fuente: `CEREBRO/DYNAMIS/sintesis/F_oferta_anclaje_backend_ltv.md`.

## Regla madre

Toda afirmación atribuible. **★ CONSENSO REAL** (2+ voces) / **◇ UNA VOZ**. Primero los ★.

> Sesgo: solo 1 de 6 fichas es de Vinícius (17%). Cluster diverso y confiable — 6 operadores independientes; los ★ son consenso real del campo.

## Modelo base

No vendas un producto: vendé una **escalera de valor al mismo avatar** (Bait $9 → Front $100-150 → Mid $200-300 → Back $400-700 → High $4-5k). El downsell es el paso anterior. La rentabilidad real está en el **backend HT** (CAC cero), no en el front. La palanca #1 de conversión es el **anclaje**: instalá el valor alto antes de revelar el precio.

---

## Lógica de decisión (MAPA DE TENSIONES)

| Decisión | Condición de decisión |
|---|---|
| **Vitalicio vs. recurrencia** | Vitalicio solo de productos maduros que ya no captan nuevos |
| **Descuento % vs. margen** | Combo/vitalicio con anclaje > bajar precio a secas |
| **ROAS front-end vs. backend** | Break-even en front solo si tenés backend armado |
| **Anclaje alto vs. credibilidad** | El anclaje debe ser real y verificable |
| **Combo simple vs. completo** | Ambos, sin mezclar ecosistemas horizontales |

## Arsenal operativo

### Escalera de valor (Vinícius)
Bait→Front→Mid→Back→High, mismo avatar en etapas distintas. Downsell = paso anterior.

### Anclaje por terceros / subasta en vivo (Mateus)
1. Listar valor acumulado. 2. "¿Cuál es un valor justo? Léanmelo." 3. Leer solo precios altos. 4. "Podría costar $5.000." 5. "Pero lo dejamos en solo $[real]."

### Combo con anclaje real (Ladeira)
Precio individual real + renovación = ancla → combo/vitalicio reducido. Verificable en la web. Sin descuentos %; escasez de tiempo. Landing solo email+teléfono.

### Oferta aniquiladora (Mateus)
Transformación alta + esfuerzo percibido mínimo + probabilidad alta (testimonios). Sin relleno. Congruencia anuncio→clase→producto.

### Backend Done-for-You tras el checkout (Pancho)
Video de escasez del backend HT ($3-5k) justo tras la compra del core ("abrí 7 plazas, agendá Zoom en 3 días"). Bonos de acción rápida distintos cada día. Entregables tangibles → refund <2%. Automatizá con IA.

### Inteligencia de oferta (Ana)
IA sobre el Excel de ventas → LTV/CAC/recompra por producto. Combo Básico (masa 80%) + Completo (fans 10%). Vitalicio solo de maduros. Reactivá base con API oficial de WhatsApp.

### Contraste de precios (Victor)
Mantené un HT visible aunque venda poco: hace barato el MT y multiplica sus ventas.

## Benchmarks

- Escalera ref.: Bait $9 / Front $100-150 / Mid $200-300 / Back $400-700 / High $4-5k (Vinícius).
- Backend aporta +35% a la facturación; entregable tangible → refund <2%; cierre de upsell 30% (Pancho).
- Narrativa 10x: $2.1M vs $148k previo; conv 1.8%; arrastre 74%/84% (Mateus).
- BF sobre base caliente: hasta 17% conv (Ladeira); ROAS 40+ con API oficial (Ana).

Fuente limpia, sin `[¿?]`. Números de Black Friday = base caliente; NO proyectar a frío. Normalizá monedas.
